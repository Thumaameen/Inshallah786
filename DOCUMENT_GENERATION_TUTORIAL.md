# DHA Document Generation Tutorial

## How to Generate and Validate Official Documents

### Step 1: Upload Your Passport
1. Open the DHA Digital Services app on your phone
2. Tap "Generate New Document"
3. Choose "Take Passport Photo" or "Upload Passport"
4. Ensure good lighting and that all passport details are clearly visible
5. Take the photo or select an existing one

### Step 2: Generate Document
1. The system will automatically:
   - Scan and verify your passport
   - Check against ICAO PKD database
   - Verify with INTERPOL systems
   - Validate with SAPS database
   - Create a secure DHA document

### Step 3: Download the Document
1. Once generated, you'll receive:
   - A secure PDF document
   - A unique Document ID
   - A QR code for validation
   - A mobile-friendly download link

For iOS devices:
- Tap "Download PDF"
- The document will open in Safari
- Tap the share button (box with arrow)
- Select "Save to Files" or "Books"

For Android devices:
- Tap "Download PDF"
- The document will be saved to Downloads
- Open with your preferred PDF viewer

### Step 4: Validate Your Document
1. Online Validation:
   - Visit https://validate.dha.gov.za
   - Enter your Document ID
   - Or scan the QR code

2. Official Verification:
   - Any DHA office can verify the document
   - Police stations can verify via SAPS system
   - Border control has direct verification access
   - Embassies can verify via ICAO system

### Security Features
Your document includes:
- Digital watermark
- QR validation code
- Holographic elements
- Microprinting
- UV security features
- Digital signatures
- Secure tracking
- Real-time validation

### Important Notes
- Documents are automatically registered in the DHA system
- Each document has a unique global identifier
- Real-time validation is available 24/7
- Documents can be verified offline via security features
- All documents are tracked in the national database
